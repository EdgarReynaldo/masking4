Documentation

A simple dialog with an OpenGL context. This is similar to a GLViewport except that it is a dialog and should typically be used as the master dialog of an OpenGL application.

void MsgDraw()
Draws the contents of the dialog. Does all the same things as GLViewport::MsgDraw() except that when it calls Draw(Bitmap &canvas) it passes a valid bitmap you can draw to with the regular Allegro drawing functions. Note however that some of those functions are not supported and others can be very slow.

This class has no child classes.
