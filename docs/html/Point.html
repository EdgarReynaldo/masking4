Operators for doing simple vector arithmetic with two points.

Examples: 

      Point a(10, 20);
      Point b(30, 40);


      Point c = a + b;    c = (40, 60)
      Point d = b - a;    d = (20, 20)
      c += a;             c = (50, 80)
      c -= b;             c = (20, 40)
