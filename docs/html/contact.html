Contact info

If you need to contact the developers of MASkinG for any reason, be it to report a bug, request a feature, ask for advice or anything else, you can do so by sending an email to miranamon@users.sourceforge.net.

You can also often find me at the Allegro.cc forums. Send me a private message or post a question on the forums. If I'm not there to answer it immediately, someone else is bound to know the answer.

MASkinG is constantly being developed on, features are being added, bugs are being fixed and so on, so to be sure you have the latest version, check the MASkinG homepage and look for updates.
