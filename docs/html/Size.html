class MASKING_DLL_DECLSPEC Size

Almost exactly the same as Point.

Documentation

Almost exactly the same as Point. The only difference is that instead of x and y coordinates this class has w and h values representing width and height and there are no operators for checking containment in a rectangle.

This class has no child classes.
